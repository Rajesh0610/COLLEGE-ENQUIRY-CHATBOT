# College Enquiry Chatbot

This project is a chatbot that helps students get information about college admissions, courses, fees, and facilities.

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js

## Features
- Answers student questions automatically
- Uses FAQ database
- Simple web interface

## How to Run
1. Clone the repository
2. Install dependencies
3. Run the server using: node server.js
4. Open index.html in browser
